## Hướng dẫn cài đặt

## Clone dự án

```bash
$ git clone https://github.com/TuanDung1304/btl-react-ecommerce.git
```

```bash
$ cd btl-react-ecommerce
```

## Cài đặt node_modules

```bash
$ yarn
```

## Running the app - Port 5173

```bash
# watch mode
$ yarn run dev
```
